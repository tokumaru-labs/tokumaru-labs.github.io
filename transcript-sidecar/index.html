Transcript Sidecar

Chrome Extension · version 0.4.0 · Tokumaru Labs

View, copy, format, and export YouTube transcripts — right in Chrome's Side Panel.

YouTube の字幕を、Chrome の Side Panel で表示・整形・書き出し。

Key features

  • Opens in the Chrome Side Panel, next to the video.
  • Reads the available caption tracks from the current YouTube video.
  • One-click Copy full transcript, with artificial caption line breaks removed.
  • Readable copy that keeps the wording but adds paragraph breaks; export as .txt or .md.
  • Copy with timestamps, or as Markdown with the video title, URL, and caption language.
  • Click a transcript timestamp to seek the video. Switch the interface between Japanese and English at any time.

Privacy

  • No account, no analytics, no ads, and no developer-operated transcript server.
  • Requests only scripting, sidePanel, and YouTube-only host access — never <all_urls>.
  • The interface-language choice is stored in extension-origin localStorage and is never sent anywhere.
  • Caption text, titles, and URLs are shown as received and never transmitted by the extension.

アカウント・分析・広告・字幕サーバーなし。権限は scripting / sidePanel と YouTube ドメインのみで、外部送信は行いません。
