Tokumaru Midnight Cockpit

Chrome Theme · version 0.1.0 (Classic) · Tokumaru Labs

A calm, deep-navy Chrome theme built around a secret-base cockpit feel — subdued tones for long sessions, with a cyan-to-blue accent reserved for the active tab.

深いネイビーを基調にした、静かなコックピット風の Chrome テーマ。アクティブタブにだけシアン〜ブルーのアクセント。

Key features

  • Deep navy and charcoal window frame with legible text on inactive tabs.
  • A calm cyan-to-blue accent reserved for the active tab and toolbar.
  • A subdued tone that stays easy on the eyes during long sessions — no gaming-RGB look.
  • The Classic edition uses colors only — no background image.

Safety & privacy

  • No JavaScript and no HTML — the theme is a manifest.json describing colors only.
  • The manifest requests zero permissions.
  • No network communication and no tracking — nothing is collected or transmitted.

スクリプトも HTML も含まず、権限はゼロ。ネットワーク通信もトラッキングもありません。
